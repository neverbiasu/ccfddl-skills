# Code of Conduct

## Our Pledge

We want `ccfddl-skills` to be a welcoming project for researchers,
students, maintainers, and agent-tool builders. Participation should be
respectful, constructive, and focused on making the project useful.

## Expected Behavior

- Be kind and professional.
- Assume good intent while still being clear about technical problems.
- Give feedback that helps the project improve.
- Respect different research backgrounds, tooling preferences, and levels of
  experience.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Dismissive or discriminatory comments.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests,
or block participants when behavior harms the project community.

If you need to report a problem, contact the repository maintainers through
the GitHub repository's issue or discussion channels.

## Attribution
